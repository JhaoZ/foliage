About

Foliage is a developer tool used to enhance the process of version control. It allows for asynchronous control of file history. Developed by Bill Zhang, Hyeonjin Kim, Smriti Srinivasan, Marcus Cai for HackIllinois 2024.

How to use?

Clicking on a node loads the history of the node. This includes any commit messages (seen by hovering) and the actual text file data contained. Once a node is selected, the user is allowed to change the file via the text editor to add more branches to the node. A user is allowed to pick any node, ensuring that no file history is lost. This data is loaded onto an interactive hierarchal tree. Distances between connected nodes represent how different the text in those nodes are. Toggle to graph mode to see all commits and how they differ (textually) from the previous commit.

Users can save their work into text files which can be reloaded.
